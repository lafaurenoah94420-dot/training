# Nahla vs le laser — chase & combo

## Vision

Nahla traque un point laser rouge qui ziguezague sur le sol du salon. Reste dessus pour monter le combo, griffes qui rayent le parquet, Nahla grossit quand elle attrape le laser. Style arcade, arrogance féline.

## Stack

Python 3 + pygame

1. ✅ Fenêtre + salon (sol, murs) + fermeture propre
2. ✅ Nahla (sprite) + déplacement ZQSD / flèches
3. ✅ Point laser qui rebondit + accélère
4. ✅ Collision → combo + score + griffures + Nahla grossit
5. ✅ Timer 90s + meilleur combo
6. ✅ Menu + fin de partie + R pour rejouer
7. ✅ Phrases de Nahla tous les 8 combo
8. ✅ 2e laser, dash, ronron, canapé, Malik, sons, grades, record, menu diff/mode
