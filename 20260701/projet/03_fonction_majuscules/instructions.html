Fonction majuscules

fonction + paramètre texte

Le concept

Jusqu'ici tu manipulais des textes directement dans le fichier. Maintenant tu vas encapsuler cette logique dans une fonction : un bloc réutilisable que tu peux appeler avec des valeurs différentes à chaque fois.

Une fonction peut recevoir un paramètre — une valeur qu'on lui passe entre parenthèses à chaque appel. Quand ce paramètre est un texte, la fonction peut lui appliquer n'importe quelle méthode de chaîne, comme .upper().

def crier(nom):
    return nom.upper()

crier("franklin")   # → "FRANKLIN"
crier("trevor")     # → "TREVOR"

Dans GTA, les personnages se font souvent remarquer en criant leur nom dans les rues de Los Santos. Une fonction permet de faire ça pour n'importe quel pseudo sans réécrire le code à chaque fois.

Instructions

Tu codes un système de réputation de rue pour San Andreas. Quand un joueur provoque quelqu'un, son pseudo s'affiche en majuscules — comme s'il le criait.

La fonction crier_rue

Cette fonction simule un cri de rue dans le style de GTA. Elle reçoit le pseudo d'un joueur et le renvoie en majuscules.

Elle reçoit un seul paramètre qui s'appelle pseudo. C'est un texte qui représente le nom du joueur tel qu'il l'a choisi — par exemple "franklin" ou "trevor". Ce texte peut être en minuscules, en majuscules, ou un mélange des deux.

La fonction doit transformer ce pseudo en majuscules avec .upper() et retourner le résultat. Si le pseudo est un texte vide, la fonction retourne un texte vide.

Voici des exemples pour vérifier que ta logique est correcte :

crier_rue("franklin")   # → "FRANKLIN"
crier_rue("trevor")     # → "TREVOR"
crier_rue("")           # → ""

Indices

💡 Indice 1 — Structure d'une fonction

Commence par def crier_rue(pseudo): suivi d'une ligne indentée. N'oublie pas les deux-points à la fin de la ligne def, et que le corps de la fonction est décalé de quatre espaces.

💡 Indice 2 — Une seule ligne dans le corps

Tu n'as besoin que d'un return avec la méthode .upper() appelée sur le paramètre. Pas besoin de if — .upper() sur un texte vide renvoie déjà un texte vide.

def crier_rue(pseudo):
    return pseudo.upper()
