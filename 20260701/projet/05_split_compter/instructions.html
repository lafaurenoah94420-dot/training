Exercice 5 / 6 — Chaînes de caractères

Split et compter

split + len

Le concept

Parfois un texte contient plusieurs morceaux séparés par un même caractère — comme une liste de courses séparée par des virgules. La méthode .split() découpe le texte à chaque occurrence d'un séparateur et renvoie une liste de morceaux.

Une fois le texte découpé, tu peux compter combien de morceaux il contient avec len(). C'est la combinaison de deux outils que tu connais déjà, appliqués l'un après l'autre.

liste = "infanterie,cavalerie,artillerie"
unites = liste.split(",")
# unites vaut ["infanterie", "cavalerie", "artillerie"]
nombre = len(unites)   # → 3

Dans Hearts of Iron IV, l'état-major envoie parfois la composition d'une armée sous forme de texte, avec les noms des divisions séparés par des virgules. Pour savoir combien de divisions sont déployées, il faut d'abord découper, puis compter.

Instructions

Tu reçois des rapports de l'état-major au format texte. Chaque rapport liste les divisions déployées sur un front, séparées par des virgules sans espace.

La fonction compter_unites

Cette fonction compte combien de divisions sont listées dans un rapport texte.

Elle reçoit un seul paramètre qui s'appelle texte_liste. C'est un texte qui contient zéro, un ou plusieurs noms de divisions séparés par des virgules. Par exemple "infanterie,cavalerie,artillerie" contient trois divisions, et "chars" en contient une seule.

La fonction doit d'abord découper le texte avec .split(","), puis compter le nombre de morceaux obtenus avec len(), et retourner ce nombre.

Attention : si le texte est vide — c'est-à-dire qu'il ne contient aucun caractère — la fonction doit retourner zéro, car il n'y a aucune division à compter. Dans ce cas précis, ne fais pas appel à .split() : vérifie d'abord si le texte est vide avec un if.

Voici des exemples pour vérifier que ta logique est correcte :

compter_unites("infanterie,cavalerie,artillerie")   # → 3
compter_unites("chars")                               # → 1
compter_unites("")                                    # → 0

Indices

💡 Indice 1 — Deux étapes enchaînées

Tu peux stocker le résultat de split dans une variable intermédiaire, puis retourner len de cette variable. Ou bien tout mettre sur une seule ligne : return len(texte_liste.split(",")) — mais seulement quand le texte n'est pas vide.

💡 Indice 2 — Gérer le texte vide en premier

Commence la fonction par un test : si texte_liste est égal à un texte vide, retourne zéro tout de suite. Sinon, découpe et compte.

def compter_unites(texte_liste):
    if texte_liste == "":
        return 0
    morceaux = texte_liste.split(",")
    return len(morceaux)
