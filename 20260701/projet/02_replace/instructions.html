Exercice 2 / 6 — Chaînes de caractères

Remplacer dans un texte

replace

Le concept

Parfois tu veux modifier un morceau précis d'un texte sans tout réécrire à la main. La méthode .replace() cherche un passage et le remplace par autre chose. Elle prend deux arguments entre parenthèses : d'abord le texte à chercher, puis le texte de remplacement.

Important : .replace() ne modifie pas la variable d'origine. Elle renvoie un nouveau texte avec le remplacement fait. C'est comme photocopier une page et corriger la copie — l'original reste intact.

rapport = "Umbrella a fermé le labo"
corrige = rapport.replace("Umbrella", "[CENSURÉ]")
# corrige vaut "[CENSURÉ] a fermé le labo"
# rapport n'a pas changé

Dans Resident Evil, les documents classifiés passent souvent par une censure avant d'être diffusés. C'est exactement le genre de transformation que .replace() permet de faire.

Instructions

Tu décryptes des fichiers retrouvés à Raccoon City. Certains passages doivent être masqués avant d'être transmis aux STARS.

Dans main.py, crée une variable rapport qui vaut exactement ce texte : "Umbrella a perdu le contrôle du T-Virus à Raccoon City".

Crée une variable code qui vaut le texte "STARS-007" — c'est l'identifiant d'un agent.

Ensuite, crée deux nouvelles variables en utilisant .replace() :

  • rapport_censure — une copie du rapport où le mot "Umbrella" est remplacé par "[CENSURÉ]".
  • code_masque — une copie du code où la partie "007" est remplacée par "███".

Voici ce que tu dois obtenir :

rapport_censure   # → "[CENSURÉ] a perdu le contrôle du T-Virus à Raccoon City"
code_masque       # → "STARS-███"

Attention : le texte original dans rapport et code ne doit pas changer. Seules les nouvelles variables contiennent la version modifiée.

Quand tu lances python3 tester.py, tu dois voir :

✅ Umbrella remplacé par [CENSURÉ]
✅ 007 remplacé par ███
✅ le mot Umbrella a disparu du rapport

Indices

💡 Indice 1 — La syntaxe de replace

La méthode s'écrit avec un point après la variable, puis deux textes séparés par une virgule : rapport.replace("Umbrella", "[CENSURÉ]"). Le résultat doit être stocké dans une nouvelle variable — sinon il disparaît.

💡 Indice 2 — Deux lignes, deux remplacements

Tu fais la même chose deux fois, sur deux variables différentes. La structure est identique : variable source, point, replace, parenthèses avec le texte à chercher et le texte de remplacement.

rapport_censure = rapport.replace(..., ...)
code_masque = code.replace(..., ...)
