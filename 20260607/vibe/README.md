# Nahla cache-cache

Reste invisible : évite le cône de vision de Malik, cache-toi derrière les meubles.

**Lancer :**
```bash
cd 20260607/vibe && python3 main.py
```

**Contrôles**
- **ZQSD / flèches** — bouger Nahla
- **Entrée / Espace** — jouer
- **R** — rejouer
- **F** — miauler pour provoquer les humains (cooldown 8 s)
- **Échap** — quitter

**Objectif :** reste cachée le plus longtemps possible en 90 s. Malik, Maman et Kays patrouillent le grand salon — cache-toi derrière les meubles.
