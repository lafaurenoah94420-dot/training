# Nahla platformer — sauts de meuble en meuble

## Vision

Side-scroller dans l'appart : Nahla court, saute de plateforme en plateforme (canapé → étagère → frigo), récolte des croquettes, évite l'aspirateur qui patrouille. Style salon chaleureux, Nahla arrogante au centre de l'action.

## Stack

Python 3 + pygame

## Feuille de route

1. ✅ Fenêtre + salon (fond, sol, meubles en plateformes)
2. ✅ Nahla — gravité + saut (ZQSD / flèches)
3. ✅ Collisions plateformes
4. ✅ Caméra qui suit Nahla + niveau large
5. ✅ Croquettes + score HUD
6. ✅ Aspirateurs patrouilleurs + 3 vies
7. ✅ Sprite Nahla, menu, victoire/défaite, R pour rejouer
8. ✅ 3 niveaux : Salon → Cuisine → Couloir (difficulté croissante)
9. ✅ Niveau 4 — Boss Méga aspirateur (arène, saute sur la tête)

## Lancer

```bash
cd 20260605/vibe && python3 main.py
```
