# Nahla platformer

Side-scroller : sauts de meuble en meuble, croquettes, aspirateurs.

**Lancer :**
```bash
cd 20260605/vibe && python3 main.py
```

**Contrôles**
- **Q / D** ou **flèches ← →** — gauche / droite
- **Z** ou **Espace** — sauter
- **Entrée / Espace** — menu → jouer
- **R** — rejouer après victoire ou défaite
- **Échap** — quitter

**Objectif :** finir les 4 niveaux. Niveaux 1–3 : croquettes + lit. **Niveau 4 :** vaincre le Méga aspirateur (saute sur sa tête, croquettes = dégâts bonus). Les vies se conservent entre les niveaux.
