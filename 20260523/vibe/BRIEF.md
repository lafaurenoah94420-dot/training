# Bataille de Tsushima (1905)

## Vision

Jeu naval **totalement différent** du jeu terrestre d'hier : mer animée, îles du détroit, navires avec **cap/orientation**, tir de **bord** (broadside) et bonus **enfilade** (crossing the T).

## Stack

Python 3 + pygame

## Feuille de route

1. ✅ Mer + îles + flottes Japon/Russie + placement + bataille auto
2. Polish visuel (fumée, éclaboussures, noms des navires)
3. Contrôle manuel d'un navire (ordres de cap)
4. Scénario historique (flotte russe en ligne, Togo en croix)
5. Sons canon / vagues

## Lancer

```bash
cd 20260523/vibe && python3 main.py
```
