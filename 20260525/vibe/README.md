# Nahla vs l'aspirateur

Boss fight pygame dans le salon.

**Lancer :** `cd 20260525/vibe && python3 main.py`

**Contrôles**
- **Flèches** ou **Z Q S D** — se déplacer (contourne les meubles)
- **Espace** — griffe ; **contre** si l'aspirateur te touche (recul + bonus)
- **S** ou **↓** sur le canapé — se cacher
- **Croquettes dorées** — marche dessus pour soigner peur et flemme
- **R** — rejouer après victoire / défaite
- **Échap** — quitter
