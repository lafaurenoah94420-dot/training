# Nahla vs l'aspirateur — boss fight

## Vision

Combat unique dans le salon : l'aspirateur avance, Nahla recule, bondit, se cache sous le canapé. Deux jauges côté Nahla (**peur** / **flemme**), une jauge boss (**aspirateur**). Victoire en vidant la vie de l'aspirateur ; défaite si la peur explose.

## Stack

Python 3 + pygame

## Feuille de route

1. ✅ Salon + Nahla + aspirateur + jauges + combat de base
2. Attaques spéciales (crachat de poils, cri à 3h du matin)
3. Phases boss (aspirateur turbo, lumière rouge)
4. Sons + écran titre polish
5. Malik entre dans la pièce (event aléatoire)
