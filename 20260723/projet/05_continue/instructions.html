Sauter des tours

if pour ignorer

Le concept

Lua n'a pas de mot-clé continue. Pour "sauter" un tour, tu mets ton code utile dans un if : si la condition est fausse, tu ne fais rien et tu passes au tour suivant.

for _, x in ipairs(liste) do
  if x > 0 then
    total = total + x
  end
end

Instructions

GTA : somme seulement les gains positifs (ignore les pertes négatives).

Fonction sans local.

somme_positifs

Paramètre nombres. Additionne uniquement les valeurs strictement positives (> 0). Retourne le total (0 si aucune).

somme_positifs({1, -2, 3})  -- → 4

Note : 0 n'est pas strictement positif, donc on l'ignore.

Quand tu lances lua tester.lua :

✅ somme_positifs({1,-2,3,-4,5}) → 9
✅ somme_positifs({-1,-2}) → 0
✅ somme_positifs({0, 4}) → 4
