Exercice 2 / 6 — Boucles avancées

While dans une fonction

while + return

Le concept

Tu peux mettre un while dans une fonction : tu boucles, puis tu return le résultat.

Instructions

Comme le bol de croquettes de Nahla : tant que la valeur est sous le plafond, tu ajoutes un pas.

Fonction sans local.

remplir

Trois paramètres : depart, plafond, pas. Tant que depart < plafond, fais depart = depart + pas. Puis retourne depart.

remplir(20, 50, 10)  -- → 50

Quand tu lances lua tester.lua :

✅ remplir(20, 50, 10) → 50
✅ remplir(0, 30, 10) → 30
✅ remplir(50, 50, 10) → 50
