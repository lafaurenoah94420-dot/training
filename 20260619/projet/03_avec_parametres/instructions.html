Fonction avec paramètres

Le concept

Une fonction peut recevoir des valeurs qu'on lui passe entre les parenthèses. Ce sont les paramètres. Ça permet de réutiliser la même fonction avec des données différentes.

def doubler(nombre):
    return nombre * 2

doubler(5)   # → 10
doubler(12)  # → 24

Instructions

La radio du toit dans Project Zomboid diffuse une alerte par zone surveillée.

La fonction alerte_zone

Crée une fonction alerte_zone avec deux paramètres :

  • zone — un texte qui est le nom de la zone (par exemple "Garage")
  • niveau — un nombre entre un et cinq qui représente le danger

La fonction doit retourner une phrase formatée comme ceci : Zone Garage — danger 3/5 (en remplaçant le nom et le niveau par les valeurs reçues).

Utilise une f-string : f"Zone {zone} — danger {niveau}/5"

alerte_zone("Garage", 3)   # → "Zone Garage — danger 3/5"
alerte_zone("Cuisine", 1)   # → "Zone Cuisine — danger 1/5"
✅ alerte_zone('Garage', 3)
✅ alerte_zone('Cuisine', 1)
✅ alerte_zone('Toit', 5)
