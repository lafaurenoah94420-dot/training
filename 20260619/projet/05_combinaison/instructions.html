Exercice 5 / 6 — Fonctions

Deux fonctions ensemble

Le concept

Tu peux appeler une fonction depuis une autre fonction. La première fait un petit travail, la deuxième s'en sert pour décider quoi faire.

def est_majeur(age):
    return age >= 18

def peut_conduire(age):
    if est_majeur(age):
        return "Oui"
    return "Non"

Instructions

L'HUD de vie dans ton jeu de survie affiche un statut selon les points de vie du joueur.

La fonction est_critique

Elle prend un paramètre vie (un nombre). Elle retourne True si la vie est inférieure ou égale à vingt, sinon False.

La fonction statut_vie

Elle prend aussi vie. Elle appelle est_critique(vie) :

  • Si c'est critique → retourne le texte "CRITIQUE"
  • Sinon → retourne le texte "OK"
✅ est_critique(15) → True
✅ est_critique(50) → False
✅ est_critique(20) → True (seuil inclus)
✅ statut_vie(15) → CRITIQUE
✅ statut_vie(80) → OK
