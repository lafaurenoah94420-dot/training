Ta première fonction

Le concept

Une fonction, c'est un bloc de code réutilisable que tu nommes. Tu le définis avec def, puis tu l'appelles plus tard avec des parenthèses.

def dire_bonjour():
    return "Salut"

message = dire_bonjour()  # message vaut "Salut"

Ici la fonction ne prend rien entre les parenthèses — c'est une fonction sans paramètre.

Instructions
