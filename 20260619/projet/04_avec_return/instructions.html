Exercice 4 / 6 — Fonctions

return — renvoyer un résultat

Le concept

return envoie une valeur à celui qui a appelé la fonction. Sans return, la fonction ne renvoie rien d'utilisable pour la suite du programme.

def addition(a, b):
    return a + b

resultat = addition(3, 4)  # resultat vaut 7

Instructions

Joel tire sur des infectés dans The Last of Us. Tu calcules combien de balles il lui reste.

La fonction munitions_restantes

Deux paramètres :

  • chargeur — combien de balles il avait avant de tirer
  • tirees — combien de balles il a tirées

La fonction retourne combien de balles restent. Si le calcul donnerait un nombre négatif, elle retourne zéro à la place — on ne peut pas avoir moins de zéro balle.

Par exemple : s'il avait trois balles et qu'il en tire dix, le résultat doit être zéro, pas moins sept.
munitions_restantes(30, 10)  # → 20
munitions_restantes(3, 10)   # → 0

Indice outil : max(0, chargeur - tirees)

✅ munitions_restantes(30, 10) → 20
✅ munitions_restantes(5, 5) → 0
✅ munitions_restantes(3, 10) → 0 (pas négatif)
✅ munitions_restantes(0, 0) → 0
