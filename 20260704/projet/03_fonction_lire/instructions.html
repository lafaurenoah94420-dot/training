Exercice 3 / 6 — Dictionnaires

Lire avec une fonction

fonction, paramètres, return

Le concept

Lire une valeur dans un dictionnaire, c'est simple : stocks["munitions"]. Mais si tu dois le faire plusieurs fois avec des dictionnaires différents, tu vas te répéter. Une fonction encapsule cette action : elle reçoit le dictionnaire et la clé, et retourne la valeur.

C'est le même principe que pour les listes ou les nombres — une fonction reçoit des paramètres, fait un calcul, et retourne un résultat avec return.

def lire_stock(stocks, item):
    return stocks[item]

reserve = {"munitions": 12, "medkit": 2}
lire_stock(reserve, "medkit")   # → 2

Dans The Last of Us, Joel vérifie régulièrement combien il reste de munitions ou de kits de soin dans son sac. Une fonction permet de poser la même question sur n'importe quel stock.

Instructions

Joel fouille un stock de ressources dans The Last of Us. Il veut savoir combien il reste d'un objet précis sans réécrire la même ligne de code à chaque fois.

La fonction lire_stock

Cette fonction reçoit deux paramètres. Le premier s'appelle stocks — c'est le dictionnaire qui contient les ressources. Chaque clé est le nom d'un objet (un texte), chaque valeur est la quantité (un nombre). Le second s'appelle item — c'est le texte de la clé qu'on veut lire, par exemple "munitions" ou "medkit".

La fonction doit retourner la valeur associée à cette clé dans le dictionnaire. Concrètement, elle lit stocks[item] et retourne ce nombre.

Attention : cette fonction suppose que la clé existe dans le dictionnaire. Si tu lui passes un nom d'objet qui n'est pas dans le stock, Python lèvera une erreur — c'est normal pour cet exercice.

Voici des exemples pour vérifier ta logique :

sac = {"munitions": 12, "medkit": 2, "bouteille": 1}
lire_stock(sac, "munitions")   # → 12
lire_stock(sac, "medkit")      # → 2
lire_stock(sac, "bouteille")   # → 1

Indices

💡 Indice 1 — La structure de la fonction

Commence par def lire_stock(stocks, item): suivi d'une ligne indentée avec return. La fonction ne fait qu'une seule chose : lire et retourner.

💡 Indice 2 — Quoi retourner

Tu connais déjà la syntaxe pour lire une clé : stocks[item]. Il suffit de mettre ça après return.

def lire_stock(stocks, item):
    return stocks[item]
