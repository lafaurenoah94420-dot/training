Exercice 1 / 6 — Dictionnaires

Lire un dictionnaire

clé → valeur, lecture, len

Le concept

Un dictionnaire stocke des paires clé → valeur. La clé est le nom de ce qu'on cherche, la valeur est ce qu'on obtient. C'est comme un placard étiqueté : l'étiquette est la clé, le contenu derrière est la valeur.

Pour lire une valeur, on utilise la clé entre crochets : mon_dict["nom_de_la_cle"]. Pour compter combien de paires le dictionnaire contient, on utilise len(mon_dict) — ça compte le nombre de clés, pas la somme des valeurs.

soins = {"herbes": 4, "spray": 2}
nb = soins["herbes"]   # → 4
types = len(soins)     # → 2 (deux clés différentes)

Dans Resident Evil, l'inventaire de soins fonctionne souvent ainsi : chaque type d'objet a un nom (la clé) et une quantité (la valeur).

Instructions

Tu fouilles la pharmacie du commissariat dans Resident Evil. Les herbes, sprays et bandages sont rangés par type, avec une quantité pour chacun.

Dans main.py, crée un dictionnaire qui s'appelle soins. Il contient trois paires clé-valeur : la clé "spray" vaut le nombre deux, la clé "herbes" vaut le nombre quatre, et la clé "bandage" vaut le nombre un.

Ensuite, crée trois variables à partir de ce dictionnaire :

  • spray — la quantité de sprays, en lisant la valeur associée à la clé "spray"
  • herbes — la quantité d'herbes, en lisant la valeur associée à la clé "herbes"
  • nb_types — le nombre de types différents de soins dans le placard, en utilisant len sur le dictionnaire
Pourquoi len et pas une addition ? Parce que len(soins) compte combien de clés existent — ici trois types différents. Si tu additionnais les valeurs (deux plus quatre plus un), tu obtiendrais sept objets au total, ce qui est une autre information. On te demande le nombre de types, pas la quantité totale.

Voici ce que tu dois obtenir si tu affiches ces variables :

spray       # → 2
herbes      # → 4
nb_types    # → 3

Quand tu lances python3 tester.py, tu dois voir :

✅ spray vaut 2
✅ herbes vaut 4
✅ nb_types vaut 3

Indices

💡 Indice 1 — Créer le dictionnaire

Un dictionnaire se crée avec des accolades et des paires séparées par des virgules. Chaque paire a une clé (un texte entre guillemets) suivie de deux-points, puis une valeur (un nombre ici).

💡 Indice 2 — Lire une valeur précise

Pour lire une quantité, écris le nom du dictionnaire, puis entre crochets le texte exact de la clé : soins["spray"]. Pour compter les types, une seule ligne : len(soins).
