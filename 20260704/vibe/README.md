# Nahla Dodge

Esquive les objets qui tombent avec Nahla.

## Lancer (pygame)

```bash
cd /Users/noah/Desktop/Python/20260704/vibe
python3 main.py
```

## Jouer sur nahla.lol

Ouvre `index.html` ou `dodge.html` dans le navigateur (dossier `20260703/vibe/` ou `20260704/vibe/`).

## Contrôles

- Flèches, Q/S (AZERTY) ou A/D (QWERTY) : déplacer Nahla
- R : rejouer après un game over
- M : retour au menu
- Échap : retour menu / quitter
