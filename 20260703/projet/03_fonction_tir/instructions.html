Tir aléatoire

random dans une fonction, return

Le concept

On peut combiner une fonction avec un module importé. Tu importes random une fois en haut du fichier, puis tu utilises random.randint à l'intérieur d'une fonction pour renvoyer une valeur différente à chaque appel.

import random

def lancer_de():
    return random.randint(1, 6)

Dans The Last of Us, chaque tir ne fait pas exactement les mêmes dégâts — il y a une fourchette min/max.

Instructions

Joel tire sur un infecté dans The Last of Us. Selon la précision et l'arme, les dégâts varient entre un minimum et un maximum.

La fonction tirer_degats

Cette fonction simule les dégâts d'un tir. Elle reçoit deux nombres : le premier s'appelle min_dmg — c'est le dégât minimum possible. Le second s'appelle max_dmg — c'est le dégât maximum possible.

La fonction doit retourner un nombre entier aléatoire entre min_dmg et max_dmg (bornes incluses), en utilisant random.randint. N'oublie pas import random en haut du fichier.

Voici des exemples pour vérifier ta logique :

tirer_degats(10, 20)   # → un nombre entre 10 et 20
tirer_degats(30, 30)   # → toujours 30
tirer_degats(0, 0)     # → 0
