Distance arrondie

import math, math.floor

Le concept

Le module math regroupe des fonctions mathématiques utiles. Tu l'importes avec import math, puis tu appelles ses fonctions en écrivant math. devant le nom.

La fonction math.floor arrondit un nombre vers le bas : elle coupe les décimales sans jamais monter. Quarante-sept virgule huit devient quarante-sept, pas quarante-huit. Dans un jeu, c'est pratique quand le GPS affiche des mètres entiers.

import math

brut = 47.8
entier = math.floor(brut)   # → 47

Dans GTA, la distance affichée sur la minimap est souvent un nombre entier — même si le calcul réel est plus précis.

Instructions

Tu courses vers un point de livraison dans GTA. Le GPS calcule la distance exacte, mais l'affichage ne montre que des mètres entiers.

À écrire dans main.py
  • Commence par import math.
  • Crée une variable distance_brute qui vaut 47.8.
  • Crée une variable distance qui reçoit math.floor(distance_brute) — la distance arrondie vers le bas.
Pourquoi arrondir vers le bas et pas au plus proche ? Parce que math.floor coupe toujours les décimales sans jamais augmenter le nombre. C'est différent de l'arrondi classique qu'on apprend en maths.

Voici ce que tu dois obtenir :

distance_brute   # → 47.8
distance         # → 47

✅ math.floor(47.8) → distance vaut 47
✅ distance est un nombre entier (pas de décimales)
