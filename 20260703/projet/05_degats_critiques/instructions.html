Exercice 5 / 6 — Modules

Dégâts critiques

math.floor + random.randint combinés

Le concept

Parfois, un calcul de jeu mélange deux modules. La partie fixe (dégâts de base) passe par math.floor pour devenir un nombre entier. La partie aléatoire (bonus critique) passe par random.randint.

Le résultat final additionne les deux. C'est comme un headshot dans un survival horror : tu as des dégâts garantis plus un bonus imprévisible.

import math
import random

def exemple(base, bonus_min, bonus_max):
    return math.floor(base) + random.randint(bonus_min, bonus_max)

Instructions

Tu vise la tête d'un zombie dans Resident Evil. Le tir de base fait des dégâts fixes, mais un headshot ajoute un bonus aléatoire.

La fonction degats_finaux

Cette fonction calcule les dégâts totaux d'un headshot. Elle reçoit trois nombres : base (les dégâts de base, qui peuvent avoir des décimales), bonus_min (le bonus minimum) et bonus_max (le bonus maximum).

Elle doit retourner la somme de deux parties : d'abord math.floor(base) pour arrondir les dégâts de base vers le bas, puis un bonus aléatoire entre bonus_min et bonus_max avec random.randint.

L'ordre compte : arrondis d'abord la base avec math.floor, puis ajoute le bonus aléatoire. Si la base vaut quarante-sept virgule huit, la partie fixe vaut quarante-sept — pas quarante-huit.

Voici des exemples pour vérifier ta logique :

degats_finaux(47.8, 5, 15)   # → entre 52 et 62
degats_finaux(20.9, 4, 4)    # → toujours 24

N'oublie pas les deux imports en haut du fichier : import math et import random.
