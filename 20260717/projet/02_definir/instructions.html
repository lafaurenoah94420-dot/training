Ta première fonction

function / return / end

Le concept

Une fonction, c'est un petit programme que tu inventes et que tu peux relancer autant de fois que tu veux. Tu la définis une fois, puis tu l'appelles avec son nom suivi de parenthèses.

function dire_bonjour()
  return "salut"
end

dire_bonjour()  -- → "salut"

En Lua, toute fonction se ferme avec end. Le mot return envoie le résultat à celui qui a appelé la fonction.
