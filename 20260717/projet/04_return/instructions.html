Return avec plafond

return + math.min

Le concept

Souvent, une fonction ne se contente pas d'un calcul brut : elle doit aussi respecter une règle du jeu. Par exemple, la vie ne peut jamais dépasser cent. Tu combines alors ton calcul et un outil comme math.min dans le même return.

function plafonner(valeur)
  return math.min(100, valeur)
end

plafonner(80)   -- → 80
plafonner(150)  -- → 100

Instructions

Même kit de soin que tout à l'heure, mais maintenant la vie ne peut plus dépasser cent. Si le calcul dépasse, on ramène à cent.

Écris tes fonctions avec function nom(...) — sans le mot local devant. Sinon le testeur ne les trouve pas.

La fonction soigner

Cette fonction reçoit deux paramètres. Le premier s'appelle vie — les points de vie actuels. Le deuxième s'appelle soin — les points récupérés.

La fonction doit additionner ces deux nombres, puis retourner le plus petit entre ce total et cent. Comme ça, même si on soigne trop, la vie reste plafonnée.

Attention : si le total est déjà en dessous de cent, on retourne le total normal. Le plafond ne change rien dans ce cas.

Voici des exemples pour vérifier que ta logique est correcte :

soigner(60, 20)  -- → 80
soigner(90, 20)  -- → 100  (90 + 20 = 110, plafonné)
