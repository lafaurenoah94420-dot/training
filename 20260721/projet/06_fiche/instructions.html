Exercice 6 / 6 — Dictionnaires

Fiche complète

creer + get + set + a_cle

Le concept

Tu combines tout : créer une fiche, lire/écrire une clé par son nom, et tester si une clé existe.

-- Une clé absente vaut nil
if fiche["arme"] ~= nil then
  -- la clé existe
end

Instructions

Mini système de fiche personnage (Zomboid / TLOU).

Toutes les fonctions sans local.

creer

Deux paramètres : nom (texte) et vie (nombre). Retourne une table avec ces deux clés.

get

Paramètres fiche et cle (texte). Retourne fiche[cle].

set

Paramètres fiche, cle, valeur. Écrit fiche[cle] = valeur.

a_cle

Paramètres fiche et cle. Retourne true si fiche[cle] n'est pas nil, sinon false.

f = creer("Dina", 85)
get(f, "nom")       -- → "Dina"
set(f, "vie", 70)
a_cle(f, "arme")    -- → false

Quand tu lances lua tester.lua, tu dois voir :

✅ creer → nom
✅ creer → vie
✅ set vie → 70
✅ a_cle(f, "nom") → true
✅ a_cle(f, "arme") → false
