Exercice 6 / 6 — Listes avancées

Mini inventaire

append, len, f-string

Le concept

Tu as maintenant plusieurs outils pour manipuler des listes : les ajouter des éléments, compter, sommer, chercher. Il est temps de les assembler dans un petit système d'inventaire, comme dans un jeu de survie.

Une fonction peut modifier une liste reçue en paramètre et la retourner pour que l'appelant garde la version à jour. Une autre peut produire un texte de résumé avec une f-string et len.

def ajouter(inventaire, objet):
    inventaire.append(objet)
    return inventaire

def resume_inventaire(inventaire):
    return f"Inventaire : {len(inventaire)} objets"

C'est le genre de logique qu'on retrouve dans Hearts of Iron IV quand on gère des stocks de matériel : on ajoute, on compte, on affiche un résumé.

Instructions

Tu construis un mini système d'inventaire pour un camp de survivants. Deux fonctions suffisent pour ajouter du matériel et afficher un résumé.

La fonction ajouter

Cette fonction reçoit deux paramètres. Le premier s'appelle inventaire — c'est la liste qui représente le sac à dos actuel. Le second s'appelle objet — c'est le texte de l'objet à ranger, par exemple "batte" ou "hache".

La fonction doit ajouter l'objet à la fin de l'inventaire avec append, puis retourner l'inventaire modifié. Retourner la liste permet à celui qui appelle la fonction de récupérer la version à jour, même si la liste a déjà été modifiée en place.

Voici un exemple d'utilisation enchaînée :

inv = []
inv = ajouter(inv, "batte")
inv = ajouter(inv, "hache")
inv = ajouter(inv, "batte")
# inv vaut ["batte", "hache", "batte"]

La fonction resume_inventaire

Cette fonction reçoit un seul paramètre : inventaire, la liste à résumer.

Elle doit retourner un texte qui indique combien d'objets l'inventaire contient. Le format exact est important : le texte doit être "Inventaire : N objets", où N est le nombre d'éléments dans la liste, calculé avec len. Utilise une f-string pour construire ce texte.

Voici des exemples :

resume_inventaire(["batte", "hache", "batte"])   # → "Inventaire : 3 objets"
resume_inventaire([])                            # → "Inventaire : 0 objets"

Indices

💡 Indice 1 — Ajouter et retourner

Dans ajouter, appelle append sur le paramètre inventaire, puis retourne cette même liste. Deux lignes suffisent.

💡 Indice 2 — Le résumé en f-string

Pour resume_inventaire, compte les éléments avec len(inventaire) et insère le résultat dans une f-string :

return f"Inventaire : {len(inventaire)} objets"

Attention à l'orthographe exacte du texte, y compris les deux-points et le mot « objets » au pluriel.
