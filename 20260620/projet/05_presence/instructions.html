Exercice 5 / 6 — Listes avancées

Présence et position

in, index, recherche

Le concept

Dans un inventaire de survie, deux questions reviennent souvent : est-ce que j'ai cet objet ? Et s'il est là, à quelle place ?

Python permet de tester la présence avec l'opérateur in : "eau" in inventaire renvoie vrai ou faux. Pour connaître la position, on peut parcourir la liste avec un compteur d'index, ou utiliser une boucle qui retourne la première position trouvée.

inv = ["eau", "conserve", "bandage", "eau"]
"eau" in inv          # → True
"fusil" in inv        # → False
inv.index("eau")      # → 0 (première occurrence)

Dans cet exercice, tu écris toi-même ces deux vérifications sous forme de fonctions réutilisables.

Instructions

Tu fouilles le sac à dos d'un survivant avant une expédition. Tu veux savoir rapidement si un objet est présent et à quel emplacement il se trouve.

La fonction contient

Cette fonction reçoit deux paramètres. Le premier s'appelle liste — c'est l'inventaire à fouiller. Le second s'appelle objet — c'est le texte qu'on cherche, par exemple "eau" ou "fusil".

La fonction doit retourner vrai si l'objet se trouve quelque part dans la liste, et faux sinon. Tu peux utiliser une boucle avec un if, ou l'opérateur in directement.

Voici des exemples :

inv = ["eau", "conserve", "bandage", "eau"]
contient(inv, "eau")    # → True
contient(inv, "fusil")  # → False

La fonction index_de

Cette fonction reçoit les mêmes deux paramètres : liste et objet.

Elle doit retourner la position de la première occurrence de l'objet dans la liste. Le premier élément est à l'index zéro, le deuxième à l'index un, et ainsi de suite.

Si l'objet n'est pas dans la liste, la fonction doit retourner -1 — c'est une convention pour dire « introuvable », sans provoquer d'erreur.

Attention : si un objet apparaît plusieurs fois, seule la première position compte. Par exemple, dans ["eau", "conserve", "bandage", "eau"], l'eau est à l'index zéro, pas à l'index trois.

Voici des exemples :

inv = ["eau", "conserve", "bandage", "eau"]
index_de(inv, "eau")       # → 0
index_de(inv, "bandage")   # → 2
index_de(inv, "croquette") # → -1

Indices

💡 Indice 1 — Tester la présence

Pour contient, la version la plus directe est de retourner le résultat de objet in liste. Si tu préfères une boucle, parcours chaque élément et retourne True dès que tu trouves une correspondance. Si la boucle finit sans rien trouver, retourne False.

💡 Indice 2 — Trouver l'index

Pour index_de, parcours la liste avec un compteur qui part de zéro. À chaque tour, si l'élément courant est égal à objet, retourne le compteur tout de suite. N'oublie pas d'augmenter le compteur à chaque tour. Si tu arrives à la fin sans rien trouver, retourne -1.

index = 0
for element in liste:
  # compare element et objet
  # si égal → return index
  index = index + 1
return -1
