Exercice 3 / 6 — Listes avancées

Compter dans une liste

boucle for, if, compteur

Le concept

Parfois tu veux savoir combien de fois un élément précis apparaît dans une liste. Python ne propose pas d'outil magique pour ça : tu parcours la liste toi-même et tu comptes à chaque fois que tu trouves ce que tu cherches.

La technique classique : une variable compteur qui part de zéro, une boucle for qui visite chaque élément, et un if qui ajoute un à chaque correspondance.

def compter_type(liste, nom):
    compteur = 0
    for element in liste:
        if element == nom:
            compteur = compteur + 1
    return compteur

Dans Resident Evil, compter combien de zombies d'un type donné restent dans une salle, c'est la même logique.

Instructions

Tu analyses un rapport de patrouille : une liste d'infectés repérés dans une zone. Tu veux une fonction réutilisable pour compter combien de fois un type précis apparaît.

La fonction compter_type

Cette fonction reçoit deux paramètres. Le premier s'appelle liste — c'est la liste de textes à parcourir, par exemple les types d'infectés observés. Le second s'appelle nom — c'est le type que tu veux compter, par exemple "coureur" ou "Tank".

La fonction doit parcourir toute la liste avec une boucle for. À chaque élément, elle vérifie avec un if si l'élément est égal à nom. Si oui, elle augmente un compteur d'une unité. À la fin, elle retourne le nombre total de correspondances.

Si le type recherché n'apparaît pas du tout dans la liste, la fonction doit retourner zéro — pas d'erreur, juste aucune occurrence trouvée.

Voici des exemples pour vérifier que ta logique est correcte :

types = ["coureur", "Tank", "coureur", "rapide", "coureur"]
compter_type(types, "coureur")   # → 3
compter_type(types, "Tank")        # → 1
compter_type(types, "mega")        # → 0

Indices

💡 Indice 1 — Le compteur

Avant la boucle, crée une variable compteur initialisée à zéro. C'est elle que tu vas augmenter à chaque fois que tu trouves une correspondance, puis retourner à la fin de la fonction.

💡 Indice 2 — La condition

Dans la boucle, compare chaque élément au paramètre nom avec ==. Si c'est égal, ajoute un au compteur :

if element == nom:
    compteur = compteur + 1
