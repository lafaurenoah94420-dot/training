# Nahla Arena

Esquive Malik, Kays et les humains qui foncent vers Nahla pour la caresser.

## Lancer

```bash
cd /Users/noah/Desktop/Python/20260620/vibe
python3 main.py
```

## Contrôles

- **Menu** : ESPACE pour jouer
- **Jeu** : flèches ou ZQSD pour bouger Nahla
- **Griffe** : ESPACE en jeu (repousse les humains proches, cooldown ~1,5 s)
- **Game Over** : R pour rejouer, ESC pour le menu
- **Pendant la partie** : ESC pour revenir au menu

## Nouveautés

- **Coup de griffe** — repousse les humains à portée
- **Bulles de dialogue** — chaque humain dit une phrase en arrivant
- **Maman** — spawn rare, lente mais énorme hitbox
- **Sons** — miaulement, griffe, « NON » au game over
- **Record sauvegardé** dans `highscore.json`

## Objectif

Tiens le plus longtemps possible. Plus tu tiens, plus les humains arrivent vite, courent fort et sont nombreux.
