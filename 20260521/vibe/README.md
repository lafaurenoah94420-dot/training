# Feu de forêt — vibe

Simulation type Project Zomboid : grille, clic pour allumer, propagation.

## Installer pygame (une fois)

```bash
python3 -m pip install pygame
```

## Lancer

```bash
cd /Users/noah/Desktop/Python/20260521/vibe
python3 main.py
```

Fermer : croix de la fenêtre ou **Échap**.  
**R** = nouvelle carte (arbres, maisons, rochers, étangs).
