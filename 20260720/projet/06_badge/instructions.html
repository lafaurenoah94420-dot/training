Badge joueur

upper + format combinés

Le concept

Tu combines plusieurs outils : majuscules + collage pour fabriquer un badge d'affichage.

local maj = string.upper(nom)
-- puis coller maj et le niveau

Instructions

Le lobby GTA affiche un badge : nom en majuscules entre crochets, puis le niveau.

Fonction sans local devant function (tu peux utiliser local à l'intérieur pour une variable temporaire).

badge

Deux paramètres : nom (texte) et niveau (nombre). Retourne une chaîne de la forme "[JOEL] — niv.12" :

  • le nom en majuscules entre [ et ]
  • un espace, un tiret long —, un espace
  • le texte niv. collé au numéro de niveau (sans espace entre niv. et le chiffre)
badge("joel", 12)  -- → "[JOEL] — niv.12"
badge("Ellie", 5)  -- → "[ELLIE] — niv.5"

Attention : le tiret est le caractère — (tiret long), pas un simple -.

Quand tu lances lua tester.lua, tu dois voir :

✅ badge("joel", 12) → "[JOEL] — niv.12"
✅ badge("Ellie", 5) → "[ELLIE] — niv.5"
✅ badge("x", 1) → "[X] — niv.1"
