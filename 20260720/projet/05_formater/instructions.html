Formater un message

string.format ou ..

Le concept

Pour coller du texte et des nombres proprement, tu peux utiliser .. ou string.format.

nom .. " a " .. vie .. " PV"
string.format("%s a %d PV", nom, vie)

Les deux sont valides. Choisis celui que tu préfères.

Instructions

L'écran médical affiche le nom et les PV sur une seule ligne.

Fonction sans local.

statut

Deux paramètres : nom (texte) et vie (nombre). Retourne une phrase du type "Joel : 72 PV" — nom, espace, deux-points, espace, vie, espace, PV.

statut("Joel", 72)  -- → "Joel : 72 PV"

Quand tu lances lua tester.lua, tu dois voir :

✅ statut("Joel", 72) → "Joel : 72 PV"
✅ statut("Ellie", 100) → "Ellie : 100 PV"
✅ statut("Dina", 0) → "Dina : 0 PV"
