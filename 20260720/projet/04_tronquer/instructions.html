Couper une chaîne

string.sub

Le concept

string.sub extrait une partie d'un texte : du caractère de début jusqu'au caractère de fin.

string.sub("Franklin", 1, 4)  -- → "Fran"
string.sub("Leon", 1, 2)      -- → "Le"

Instructions

L'UI affiche seulement le début d'un long nom de joueur.

Fonction sans local.

debut

Deux paramètres : texte et n (un nombre). Retourne les n premiers caractères de texte (du caractère 1 jusqu'au caractère n).

debut("Franklin", 4)  -- → "Fran"

Quand tu lances lua tester.lua, tu dois voir :

✅ debut("Franklin", 4) → "Fran"
✅ debut("Leon", 2) → "Le"
✅ debut("A", 1) → "A"
