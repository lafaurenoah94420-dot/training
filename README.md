# Formation Python

Espace de travail pour apprendre Python : exercices courts, mini-projets guidés et créneaux libres, organisés par jour dans des dossiers datés (`YYYYMMDD`).

## Prérequis

- [Python 3](https://www.python.org/downloads/) installé  
- Depuis un dossier de session, les commandes sont du type :

```bash
python3 chemin/vers/le/fichier.py
```

(adapte `python3` / `python` selon ton installation.)

## Organisation du dépôt

| Élément | Rôle |
|--------|------|
| `YYYYMMDD/` | Une journée de travail (ex. `20260505/`) |
| `…/exos/` | Petits fichiers d’exercice, à lancer un par un |
| `…/build/` | Mini-projet : souvent un `main.py` fourni et une partie à compléter (ex. `project.py`) |
| `…/libre/` | Projet libre du jour |

Les récapitulatifs ou notes personnelles peuvent rester à côté (ex. `recap.md` dans le dossier du jour).

## Configuration Cursor (local)

Le dossier `.cursor/` contient règles et compétences agent pour l’environnement d’apprentissage. Tu peux ignorer ce dossier si tu ouvres le projet sans Cursor.
