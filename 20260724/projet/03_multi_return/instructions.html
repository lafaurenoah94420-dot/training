Plusieurs return

return a, b

Le concept

Une fonction Lua peut renvoyer plusieurs valeurs d'un coup, séparées par des virgules.

function duo()
  return 1, 2
end
x, y = duo()  -- x=1, y=2

Instructions

GTA : une fonction qui renvoie le plus petit et le plus grand de deux nombres.

Fonction sans local.

minmax

Deux paramètres a et b. Retourne d'abord le minimum, puis le maximum (tu peux utiliser math.min et math.max).

minmax(3, 8)   -- → 3, 8
minmax(10, 2)  -- → 2, 10

Quand tu lances lua tester.lua :

✅ minmax(3, 8) min → 3
✅ minmax(3, 8) max → 8
✅ minmax(10, 2) min → 2
✅ minmax(10, 2) max → 10
