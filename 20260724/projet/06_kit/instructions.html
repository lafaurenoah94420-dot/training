Kit joueur

table + or + min

Le concept

Tu combines défauts, tables, et plafond avec math.min.

Instructions

Deux fonctions pour un perso TLOU.

Sans local devant les fonctions.

creer

Paramètres nom et armure (optionnel, défaut 0). Retourne une table { nom = nom, armure = armure }.

soigner

Paramètres vie et soin (optionnel, défaut 20). Retourne math.min(100, vie + soin).

creer("Ellie")        -- { nom = "Ellie", armure = 0 }
soigner(80)           -- → 100
soigner(80, 5)        -- → 85
