Exercice 4 / 6 — Dictionnaires

Vérifier une clé

in, clé vs valeur

Le concept

L'opérateur in permet de tester si une clé existe dans un dictionnaire. Il renvoie vrai ou faux. C'est l'équivalent de « est-ce que j'ai cet objet dans mon sac ? » — mais attention à bien comprendre ce que Python vérifie.

Sur un dictionnaire, in cherche dans les clés, pas dans les valeurs. Si tu as un bandage avec une quantité de zéro, la clé "bandage" existe toujours : "bandage" in inventaire renvoie vrai, même si la quantité vaut zéro.

sac = {"bandage": 2, "bouteille": 1}
"bandage" in sac    # → True  (la clé existe)
"fusil" in sac      # → False (pas de clé fusil)
2 in sac            # → False (2 est une valeur, pas une clé)

Dans The Last of Us, fouiller la sacoche de Joel revient à vérifier si un type d'objet est présent — pas à compter combien il en reste.

Instructions

Avant une expédition dans The Last of Us, tu veux savoir rapidement si un objet est dans la sacoche de Joel, sans te soucier de la quantité exacte pour l'instant.

La fonction a_objet

Cette fonction reçoit deux paramètres. Le premier s'appelle inventaire — c'est le dictionnaire qui représente la sacoche. Chaque clé est le nom d'un objet (un texte), chaque valeur est la quantité (un nombre). Le second s'appelle nom — c'est le texte de l'objet qu'on cherche, par exemple "bandage" ou "fusil".

La fonction doit retourner vrai si l'objet existe comme clé dans le dictionnaire, et faux sinon. Elle ne regarde pas la quantité : elle vérifie seulement si le nom est une clé du dictionnaire.

Piège classique : in teste la présence d'une clé, pas d'une valeur. Si l'inventaire contient {"bandage": 0}, alors a_objet(inventaire, "bandage") retourne vrai — parce que la clé "bandage" existe, même si la quantité est zéro. En revanche, a_objet(inventaire, 0) retournerait faux, car zéro n'est pas une clé du dictionnaire.

Voici des exemples pour vérifier ta logique :

sac = {"bandage": 2, "bouteille": 1}
a_objet(sac, "bandage")   # → True
a_objet(sac, "fusil")     # → False
a_objet({"bandage": 0}, "bandage")  # → True (clé présente, quantité zéro)

Quand tu lances python3 tester.py, tu dois voir :

✅ bandage présent
✅ fusil absent
✅ clé avec quantité zéro → True

Indices

💡 Indice 1 — L'outil à utiliser

Tu as déjà vu in sur les listes. Sur un dictionnaire, ça fonctionne pareil : nom in inventaire retourne vrai ou faux. Ta fonction peut retourner ce résultat directement.

💡 Indice 2 — Clé, pas quantité

Ne compare pas la quantité à zéro. Ne parcours pas les valeurs. La question est simple : est-ce que nom est une clé du dictionnaire ? Une seule ligne suffit :

return nom in inventaire
