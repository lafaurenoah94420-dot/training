# Nahla vs Zombies

Tire des croquettes sur les zombies. Flèches/ZQSD pour bouger, Espace ou clic pour tirer.

## Lancer

```bash
cd /Users/noah/Desktop/Python/20260621/vibe
python3 main.py
```

- ZQSD / flèches — bouger Nahla
- Souris — viser (viseur + ligne orange)
- Clic gauche maintenu ou Espace — tirer vers la souris

## Objectif

Tue assez de zombies pour passer à la vague suivante. Chaque vague : plus de zombies, plus rapides.
