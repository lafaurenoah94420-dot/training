# Nahla vs Zombies — pygame shooter

Nahla au centre. Les zombies avancent depuis les bords. Flèches/ZQSD pour bouger, Espace pour tirer des croquettes. Vagues de plus en plus dures.

**Stack :** Python + pygame

## Feuille de route

1. ✅ Fenêtre + Nahla déplaçable + fond
2. ✅ Zombies qui spawnent et foncent vers Nahla
3. ✅ Tir de croquettes (Espace) + dégâts
4. ✅ Vies, score, vagues
5. ✅ Game Over + R pour rejouer
6. Polish (sons, particules, boss…) — si Noah demande
